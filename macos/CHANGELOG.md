- 适配新版 Codex 首页 DOM：移除会把顶部占位区拉满主窗口的旧 `100%` 高度规则，快捷操作回到真实输入框上方，不再出现中央巨大空白或按钮沉底
- 实时验收新增 QQ 首页几何约束，快捷操作必须完整位于输入框上方且输入框不得超出主工作区
- 兼容新版 Codex 在部分首页状态完全卸载侧栏 DOM 的行为：renderer 仍由官方 `app://` 主区与真实输入框共同识别，不再因此误报“启动失败”或阻断原皮/主题往返
- 新版 Electron 已不提供可用的 CDP 窗口 ID，而后台 watcher 调用 `AXZoomWindow` 又会触发 macOS 辅助功能授权；因此删除会静默失败的合成窗口按钮命中区，右上三键仅保留原型装饰，拖动、缩放、最小化、最大化与关闭统一使用 Codex 原生 macOS 窗口控件
- 主题 chrome 增加结构版本迁移；覆盖安装同版本引擎时也会替换旧 DOM，不再残留已经删除的按钮或其它旧组件
- 实时验收对主题切换瞬间尚未挂载的 QQ 主区增加空值保护，不再因过渡帧抛异常或假报切换失败
- QQ 响应式验收同步兼容新版首页可选侧栏，侧栏卸载时不再读取空几何
